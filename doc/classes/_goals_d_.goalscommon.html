Class GoalsCommon

公共储存区域，它包含了一些公共属性。

此类可自定义扩展属性，但是它会有一些本身就存在的值，自定义时可不要把它们覆盖了哦！

HASH_ENCRYPT

HASH_ENCRYPT: boolean
  • Defined in goals.d.ts:485

是否加密uri中的hash值

加密会使用chaotic加密，加密后的字符长度会大量变长，使用加密的用作在于更好的保护你的uri地址

默认为false

它是只读的，你可以在初始化的时候设置它

HASH_PARAMS_DELIMITER

HASH_PARAMS_DELIMITER: string

PAGE_MAXINUM

PAGE_MAXINUM: number
  • Defined in goals.d.ts:496

最大可缓存多少个页面，页面缓存可大幅度提升页面二次加载速度

值说明：

-1表示无限制

0表示不缓存

默认为10
